# Branding Standards

InfraLynx documentation and product experience should project a disciplined enterprise identity with restrained use of color and high readability.

## Branding and UX Standards

Primary color:

- InfraLynx Navy `#2A3F5F`

Palette:

- Background `#1E2A38`
- Surface `#3A4A5F`
- Text `#FFFFFF`
- Secondary text `#C7CED6`
- Accent `#E6E1D9`
- Borders `#4A5A6A`

## Brand Direction

- Dark-first presentation
- High contrast readability
- Minimal accent usage
- Functional, structured layout over decorative treatment

## Documentation Application

The MkDocs theme uses Material dark mode with custom CSS overrides so documentation remains maintainable while still aligning with InfraLynx visual standards.
