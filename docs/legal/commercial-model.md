# Commercial Model

InfraLynx follows an Apache 2.0 open-core foundation with room for commercial expansion.

## Public Foundation

- public repositories under Apache 2.0
- open documentation and architecture records
- contribution intake through GitHub workflows

## Commercial Flexibility

InfraLynx retains the option to provide:

- hosted SaaS offerings
- commercial support
- enterprise integrations
- premium operational tooling
- managed deployment services

## Contributor And IP Position

Contributor terms are structured so InfraLynx can reuse and relicense accepted contributions as needed for commercial offerings while keeping the public repositories under Apache 2.0.

## Trademark Ownership

InfraLynx names, marks, and brand assets remain controlled by the project owner even where source code is openly licensed.
