InfraLynx public repositories are released under the Apache License, Version 2.0.

## Open Core Position

The public codebase is intended to be usable, inspectable, and redistributable under Apache 2.0. This supports adoption, integration, and contribution without imposing copyleft obligations.

## Repository Requirements

- Each repository includes `LICENSE`, `NOTICE`, `VERSION`, and `CHANGELOG.md`.
- README files link to licensing and contribution material.
- Contributions are accepted under the contributor terms in `CLA.md`.
