# Product Overview

InfraLynx is an enterprise-grade infrastructure platform intended to replace NetBox with a more modular, maintainable, and extensible architecture.

## Platform Intent

InfraLynx combines infrastructure source-of-truth capabilities with a modern application architecture designed for strict governance, multi-team contribution, and long-term product evolution.

## Domain Coverage

InfraLynx is planned across the following mandatory domains:

- Core platform: authentication, RBAC, tenants, tagging, statuses, audit logging, notifications
- DCIM: sites, racks, devices, modules, ports, power, cabling
- IPAM: VRFs, prefixes, IP addresses, VLANs, allocation logic
- Network modeling: interfaces, L2 and L3 relationships, topology, path tracing
- Virtualization: clusters, hypervisors, VMs
- Automation: jobs, imports, exports, integrations, webhooks

## Product Principles

- Enterprise-first governance and maintainability
- Modular domain boundaries
- Multi-database support with PostgreSQL as primary and SQL Server and MariaDB as supported engines
- Documentation and ADR discipline from the start
